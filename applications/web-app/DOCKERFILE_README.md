# Dockerfile для web-app

## Описание

Простейший Dockerfile для сборки и запуска фронтенд приложения на Nuxt.js.

## Использование

### Сборка образа

```bash
cd applications/web-app
docker build -t web-app .
```

### Запуск контейнера

```bash
docker run -p 80:80 web-app
```

Приложение будет доступно на http://localhost:80

## Особенности

- Использует multi-stage build для оптимизации размера образа
- На основе Node.js 18 Alpine для сборки
- Финальный образ на основе Nginx Alpine для продакшена
- Автоматически собирает Nuxt.js приложение
- Настроен nginx для корректной работы SPA

## Требования

- Docker должен быть установлен
- Порт 80 должен быть свободен (или используйте другой порт при запуске)
