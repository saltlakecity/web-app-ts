# Web App (Nuxt 4)

Это приложение для работы с формами, перенесённое из Vue 3 + Vite в Nuxt 4.

## Структура проекта

```
app/
├── components/          # Vue компоненты
│   ├── forms/          # Компоненты для работы с формами
│   └── form-view/      # Компоненты для просмотра форм
├── composables/        # Vue composables
├── lib/               # Утилиты и API клиенты
├── types/             # TypeScript типы
├── assets/            # Статические ресурсы
│   └── css/           # CSS стили
└── app.vue            # Главный компонент приложения
```

## Установка

```bash
# Установка зависимостей
pnpm install

# Копирование переменных окружения
cp env.example .env

# Запуск в режиме разработки
pnpm dev
```

## Переменные окружения

Создайте файл `.env` на основе `env.example`:

```env
NUXT_PUBLIC_API_URL=http://localhost:3001
```

## Особенности миграции

1. **Структура Nuxt 4**: Все файлы находятся в папке `app/`
2. **Импорты**: Используются относительные пути вместо алиасов
3. **Конфигурация**: Настроена в `nuxt.config.ts`
4. **Стили**: Вынесены в отдельный CSS файл
5. **Компоненты**: Автоматически регистрируются в папке `components/`

## Компоненты

- **FormsList**: Список доступных форм
- **Form**: Карточка отдельной формы
- **FormView**: Просмотр и заполнение формы

## Composables

- **useForms**: Работа с формами и полями
- **useTelegramAuth**: Аутентификация через Telegram

## API

Используется tRPC для связи с бэкендом. Конфигурация в `lib/trpc.ts` и `lib/trpc-api.ts`.
